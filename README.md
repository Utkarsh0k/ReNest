ReNest is a modern web-based marketplace designed exclusively for college students to buy and sell second-hand items within their campus. It provides a simple, secure, and user-friendly platform for exchanging books, electronics, furniture, cycles, accessories, and other reusable products.

Built as a **Summer Internship Project**, ReNest focuses on clean UI, responsive design, and complete CRUD functionality using **HTML, CSS, JavaScript, and Local Storage**.

# рџ“– How It Works

### Authentication

* Users can register and log in.
* Authentication is managed using Local Storage.

### Selling Items

* Upload an image using Drag & Drop or Browse.
* Enter product details.
* Publish the listing.

### Marketplace

* Browse all available products.
* Search by keyword.
* Filter by category.
* View complete product details.

### Dashboard

* View your own listings.
* Mark products as Sold.
* Delete listings.

### Profile

* View account information.
* Display total listings.
* Display total listing value.
